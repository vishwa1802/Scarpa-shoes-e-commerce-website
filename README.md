# 👟 Scarpa Shoes – E-commerce Website

Scarpa Shoes is a modern e-commerce web application developed to provide users with a seamless online shopping experience for footwear. The platform allows customers to explore a variety of stylish shoes, view product details, add items to the cart, and complete purchases through a responsive and user-friendly interface.

The website features a clean UI and smooth navigation, ensuring a great browsing experience across different devices. This project demonstrates modern web development practices including component-based architecture, responsive design, product catalog management, and shopping cart functionality.

---

## 🚀 Features

- Product catalog with detailed product pages
- Shopping cart functionality
- Secure checkout process
- User authentication (Login & Signup)
- Admin panel for product management
- Responsive design for mobile and desktop
- Modern UI with reusable components

## ▶️ Installation & Setup

Clone the repository

git clone https://github.com/vishwa1802/Scarpa-Shoes-E-commerce-Website.git

Go to project folder

cd Scarpa-Shoes-E-commerce-Website

Install dependencies

npm install

Run development server

npm run dev

Open in browser

http://localhost:5173
